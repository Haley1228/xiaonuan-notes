# 1. AAAI 2025
**Towards Multimodal Sentiment Analysis via Hierarchical Correlation Modeling with Semantic Distribution Constraints**<br>
**通过具有语义分布约束的分层相关建模实现多模态情感分析**

## intorduciton

通过利用各种数据模式（例如文本、视频和音频），情感分析正在迅速发展。然而，大多数现有技术只学习反映强相关性的原子级特征，而忽略了多模态数据中更复杂的组合。此外，他们还忽视了模态之间语义分布的不协调。有鉴于此，文章引入了一种新的分层相关建模网络 （HCMNet），它通过探索基于动态注意力推理的原子级相关性和通过拓扑图推理的成分级相关性来增强多模态情感分析。此外，文章还从原子水平和组成水平的角度减轻了模态之间分布不一致的影响。具体来说，文章首先设计了一个原子级的对比损失，它约束了跨模态的语义分布，以减轻原子级的不一致。然后，文章设计了一个图最优传输模块，将不同图的传输流集成在一起，以约束组合层的语义分布，从而减少组合节点的不一致。在三个公共基准数据集上的实验表明，所提出的模型优于最先进的方法
## Motivation

现有MSA方法的两大局限：
- “只见树木不见森林”：仅捕捉原子级强关联（如单帧画面与单词），无法建模组合级弱关联（如多帧画面的时空依赖、文本上下文情感递进）。
- “语义鸿沟”问题：不同模态数据（如视频与文本）的情感语义分布不一致，直接融合会导致信息模糊，影响分类精度。
## Ctribution

- 文章表示HCMNet是第一个利用文本和视觉模态之间的层次语义相关性来联合建模 MSA 任务的原子级和组合级相关性的工作。
- 文章还通过对比学习和最优传输学习，从原子级和组合级的角度减轻了模态间分布不一致的影响。
- 同时，HCMNet 方法的通用性也提供了将其扩展到其他多模态理解任务的可能性

# 2.# AAAI 2026
 **Cross-modal Prompting for Balanced Incomplete Multi-modal Emotion Recognition**
 
 **用于平衡不完整多模态情感识别的跨模态提示**
## Introduciton

 多模态情感识别（MER）能处理讽刺、隐喻等复杂场景，广泛应用于抑郁症检测、对话系统等领域，但传统MER假设所有模态数据完整，而现实中数据收集、传输等限制导致模态缺失，形成不完整多模态情感识别（IMER）问题，破坏跨模态一致性，降低识别精度。现有方法或通过生成模型恢复缺失数据，或学习统一表征，但均未解决IMER中模态不平衡的核心痛点。针对此问题，本文提出跨模态提示学习（ComP）方法，通过增强模态特定特征与跨模态一致信息，提升各模态性能及整体识别准确率。
## Motivation

核心研究问题是IMER中的模态不平衡问题：
1. 一是“模态之间性能差距”（不同模态原生表现差异显著）
2. 二是“模态欠优化”（多模态协同训练后部分模态性能反而下降），根源在于多模态数据的内在异质性（部分模态任务相关信息不足、特征空间冲突）。现有方法要么忽略该差异，要么仅独立处理各模态后融合，未能有效利用跨模态互补信息。
针对这一问题，本文提出通过模态之间的信息传递强化模态之间的协同作用，从而在提升单个模态性能的前提下提升模型整体性能。这一方法解决两个关键问题：（1）应向其他模态传递何种知识？（2）各模态如何融合自身与跨模态知识？因此，本文旨在通过“提示生成+知识传播”，强化跨模态情感相关一致信息，抑制误导性信息，充分激活非主导模态潜力
